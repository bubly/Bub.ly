About bub.ly

Bub.ly

Bub.ly is here to provide all your link-shortening and file-uploading needs. You can use our bookmarklet to shorten links on the web, and use either our uploading form/droplet for your Mac OSX Dock.

You can even use your own custom domain instead of bub.ly. We also have an API so you can make your own applications!

API

Bub.ly provides both unauthenticated and an authenticated API to shorten links and upload files.

Shorten Links

All requests to shorten links are GET.

        http://bub.ly/shorten_url?url=http://google.com/
         -> returns a html page with link selected.

Make an AJAX request to the shorten_url to receive JSON

        http://bub.ly/shorten_url?url=http://google.com/
         ->  {'url': 'http://bub.ly/atc/'}

To shorten links via an authenticated user. Click here to receive your API key.

        http://bub.ly/shorten_url?url=http://google.com/&api_key=xxxxxxxxxx
Upload Files

All requests to upload files are POST. The trailing slash to the url is important as redirects does not carry POST data forward.

        curl -d "api_key=xxxx&file=@filename.jpg" http://bub.ly/file_upload/
